# Manuals

Full-length reference manuals, organised into parts and chapters. Unlike [Articles](../articles/index.md) (single-topic write-ups) or [Projects](../projects/index.md) (a build narrative read start to finish), manuals are reference works — jump straight to the chapter you need, or read a part in sequence. Each one has its own chapter nav in the sidebar, with Material's prev/next footer links to move through it in order.
